# Coinbook

## Project Overview
Coinbook is a comprehensive application designed to manage and catalog coin collections. It provides features for adding, editing, and viewing coins with detailed information, making it perfect for numismatists and collectors alike.

## Features
- Add new coins with essential details
- Edit existing coin entries
- Search and filter coins in the collection
- User-friendly interface for managing collections
- Export collection data

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Heroku

## Installation
To get started with Coinbook, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/tyagi2607/bookopedia.git
cd bookopedia
npm install
```

## Usage
To run the application in development mode:

```bash
npm start
```

Visit `http://localhost:3000` in your browser to see the app in action.
